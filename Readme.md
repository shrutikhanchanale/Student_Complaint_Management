# Student Complaint Management System

A Python-based system for managing and tracking student complaints in educational institutions.

## Features

- Submit and track student complaints
- Admin dashboard for complaint management
- Status tracking and updates
- Secure authentication system
- Email notifications
- Report generation

## Installation

```bash
git clone https://github.com/yourusername/studentComplaint
cd studentComplaint
pip install -r requirements.txt
```

## Usage

```bash
python app.py
```

## Configuration

1. Create a `.env` file
2. Add your configuration:
```
DB_HOST=localhost
DB_USER=user
DB_PASS=password
```

## Tech Stack

- Python 3.x
- SQLite/MySQL
- Flask Framework
- Bootstrap
